spell word(s) using a spelling alphabet.

== Synopsis

	spell [-hl] <word(s)>

== Examples

To set a default language you may use an alias:

	alias spell="spell -l de"

== Options

*-h* :: Print this usage note (Default: false)
*-l* string:: Spelling alphabet to use (Default: en)
